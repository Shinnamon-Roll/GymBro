บทนี้อธิบายวิธีการใช้งานระบบ GymBro Management System อย่างละเอียด แบ่งตามสิทธิ์การใช้งานของผู้ใช้ 3 กลุ่มหลัก ได้แก่ ผู้ใช้งานทั่วไป (Public), สมาชิก (Member), และผู้ดูแลระบบ (Admin)

\section{ส่วนผู้ใช้งานทั่วไป (Public Section)}
ส่วนนี้สำหรับผู้ที่ยังไม่ได้เข้าสู่ระบบ สามารถใช้งานได้ทุกคน

\subsection{การใช้งานหน้าเข้าสู่ระบบ (Login Page)}
หน้าแรกของระบบที่ใช้สำหรับยืนยันตัวตนก่อนเข้าใช้งาน

\subsubsection{ขั้นตอนการเข้าสู่ระบบ}
\begin{enumerate}
    \item เปิดเว็บเบราว์เซอร์และเข้าสู่หน้าเว็บไซต์ ระบบจะแสดงแบบฟอร์ม Login
    \item กรอก \textbf{Email Address} ที่ได้ลงทะเบียนไว้
    \item กรอก \textbf{Phone Number} (ใช้แทนรหัสผ่าน)
    \item กดปุ่ม \textbf{SIGN IN}
    \item หากข้อมูลถูกต้อง ระบบจะตรวจสอบสิทธิ์ (Role) และเปลี่ยนหน้าไปยัง Dashboard ที่เหมาะสม (User หรือ Admin)
    \item หากข้อมูลไม่ถูกต้อง ระบบจะแสดงข้อความแจ้งเตือนสีแดง "Invalid credentials"
\end{enumerate}

    \caption{หน้าจอเข้าสู่ระบบ (Login)}

\subsection{การใช้งานหน้าลงทะเบียน (Register Page)}
สำหรับผู้ใช้งานใหม่ที่ต้องการสมัครสมาชิก

\subsubsection{ขั้นตอนการสมัครสมาชิก}
\begin{enumerate}
    \item ในหน้า Login ให้กดลิงก์ "Don't have an account? Register here"
    \item ระบบจะพาไปที่หน้า Register
    \item กรอก \textbf{Full Name} (ชื่อ-นามสกุล)
    \item กรอก \textbf{Email Address} (ห้ามซ้ำกับที่มีในระบบ)
    \item กรอก \textbf{Phone Number} (ใช้สำหรับล็อกอินในภายหลัง)
    \item กดปุ่ม \textbf{CREATE ACCOUNT}
    \item หากสำเร็จ ระบบจะแจ้งเตือนและพากลับไปหน้า Login เพื่อให้เข้าสู่ระบบใหม่
\end{enumerate}

\section{ส่วนสมาชิก (Member Section)}
ส่วนนี้สำหรับผู้ใช้งานที่ล็อกอินด้วยสิทธิ์สมาชิก (User)

\subsection{การใช้งานหน้าแดชบอร์ดสมาชิก (User Dashboard)}
หน้าหลักของสมาชิก ใช้สำหรับดูตารางฝึกซ้อมและจองเซสชัน

\subsubsection{การดูตารางฝึกซ้อม}
\begin{enumerate}
    \item เมื่อเข้าสู่ระบบสำเร็จ จะพบตารางแสดงรายการจองทั้งหมดในวันนี้ (All Bookings)
    \item รายการที่เป็นของตนเอง (My Session) จะถูกไฮไลต์ด้วยสีพื้นหลังที่แตกต่างเพื่อให้สังเกตได้ง่าย
    \item ตารางแสดงข้อมูลเวลา, กิจกรรม/อุปกรณ์, และชื่อเทรนเนอร์
\end{enumerate}

\subsubsection{การจองเซสชันใหม่ (Booking Session)}
\begin{enumerate}
    \item กดปุ่ม \textbf{BOOK SESSION} ที่มุมขวาบน
    \item ระบบจะแสดงหน้าต่าง (Modal) สำหรับกรอกข้อมูลการจอง
    \item เลือก \textbf{Date} (วันที่), \textbf{Time} (เวลา), และ \textbf{Duration} (ระยะเวลา)
    \item ระบบจะตรวจสอบความว่าง (Availability) ของเทรนเนอร์และอุปกรณ์โดยอัตโนมัติ หากไม่ว่างตัวเลือกนั้นจะถูกปิดการใช้งาน (Disabled)
    \item เลือก \textbf{Trainer} และ \textbf{Equipment} ที่ต้องการ
    \item กดปุ่ม \textbf{CONFIRM BOOKING}
    \item ระบบจะบันทึกข้อมูลและรีเฟรชตารางแสดงผลทันที
\end{enumerate}

\subsubsection{การยกเลิกการจอง (Cancel Session)}
\begin{enumerate}
    \item ในตารางรายการจอง ค้นหารายการที่เป็นของตนเอง
    \item กดปุ่ม \textbf{Cancel} สีแดงที่อยู่ในคอลัมน์ Status/Action
    \item กดยืนยันในหน้าต่าง Confirm Dialog
    \item ระบบจะลบรายการจองนั้นออกจากฐานข้อมูล
\end{enumerate}

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.8\textwidth]{images/placeholder.png}
    \caption{หน้าแดชบอร์ดสมาชิกและหน้าต่างการจอง (User Dashboard)}
    \label{fig:user_dashboard}
\end{figure}

\subsection{การใช้งานหน้าข้อมูลส่วนตัว (My Profile)}
หน้าแสดงรายละเอียดข้อมูลสมาชิก

\subsubsection{ขั้นตอนการดูข้อมูล}
\begin{enumerate}
    \item ที่แถบเมนูด้านซ้าย เลือกเมนู \textbf{My Profile}
    \item ระบบจะแสดงข้อมูลชื่อ, อีเมล, เบอร์โทรศัพท์, ประเภทสมาชิก (Member Type), และวันหมดอายุสมาชิก
\end{enumerate}

\section{ส่วนผู้ดูแลระบบ (Admin Section)}
ส่วนนี้สำหรับผู้ใช้งานที่ล็อกอินด้วยสิทธิ์ผู้ดูแลระบบ (Admin)

\subsection{การใช้งานหน้าแดชบอร์ดผู้ดูแลระบบ (Admin Dashboard)}
หน้าสรุปภาพรวมของระบบ

\subsubsection{การดูสถิติและตารางงานวันนี้}
\begin{enumerate}
    \item แสดงจำนวนรวมของลูกค้า (Customers), เทรนเนอร์ (Trainers), อุปกรณ์ (Equipments), และเซสชันวันนี้ (Today's Sessions)
    \item ตารางด้านล่างแสดงรายการจองทั้งหมดที่เกิดขึ้นในวันนี้แบบ Real-time
\end{enumerate}

\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.8\textwidth]{images/placeholder.png}
    \caption{หน้าแดชบอร์ดผู้ดูแลระบบ (Admin Dashboard)}
    \label{fig:admin_dashboard}
\end{figure}

\subsection{การจัดการข้อมูลลูกค้า (Customer Management)}
เมนู \textbf{Customers} สำหรับบริหารจัดการข้อมูลสมาชิก

\subsubsection{การเพิ่มลูกค้าใหม่}
\begin{enumerate}
    \item กรอกข้อมูลในฟอร์มด้านบน: Name, Email, Phone
    \item เลือก \textbf{Member Type} (เช่น Monthly) และ \textbf{Level} (เช่น Silver)
    \item ระบุ \textbf{Start Date}
    \item กดปุ่ม \textbf{SAVE CUSTOMER}
\end{enumerate}

\subsubsection{การแก้ไขและลบข้อมูล}
\begin{enumerate}
    \item \textbf{แก้ไข:} กดปุ่ม \textbf{EDIT} ในแถวที่ต้องการ ระบบจะแสดง Popup (Prompt) ให้กรอกข้อมูลใหม่ทีละค่า (ชื่อ, อีเมล, เบอร์โทร)
    \item \textbf{ลบ:} กดปุ่ม \textbf{DELETE} สีแดง และกดยืนยันเพื่อลบข้อมูลออกจากระบบถาวร
\end{enumerate}

\subsubsection{การเพิ่มเทรนเนอร์}
\begin{enumerate}
    \item กรอกข้อมูลในฟอร์ม: Name, Specialty (ความถนัด), Phone
    \item เลือก \textbf{Level} (Junior, Senior, Master)
    \item กดปุ่ม \textbf{SAVE TRAINER}
\end{enumerate}

\subsection{การจัดการอุปกรณ์ (Equipment Management)}
เมนู \textbf{Equipment} สำหรับบริหารจัดการอุปกรณ์ในยิม

\subsubsection{การเพิ่มและจัดการสถานะอุปกรณ์}
\begin{enumerate}
    \item เพิ่มอุปกรณ์ใหม่โดยระบุชื่อและหมวดหมู่
    \item สามารถกำหนดสถานะ (Status) ได้ว่าเป็น \textbf{Available} (พร้อมใช้) หรือ \textbf{Maintenance} (ซ่อมบำรุง)
    \item หากอุปกรณ์สถานะเป็น Maintenance ระบบจะไม่ให้สมาชิกจองใช้อุปกรณ์นั้น
\end{enumerate}

    \item กดปุ่ม \textbf{BOOK NOW} ระบบจะตรวจสอบความขัดแย้งของเวลาและบันทึกข้อมูล

    \item สามารถกรองดูข้อมูลตามวันที่ได้โดยเลือกวันที่ที่ \textbf{Filter Date} และกด Apply

\section{ฟีเจอร์ใหม่และการใช้งานเพิ่มเติม (New Features \& Usage)}
ระบบได้มีการพัฒนาฟีเจอร์เพิ่มเติมเพื่ออำนวยความสะดวกในการใช้งาน ดังนี้

\subsection{การใช้งานโหมดมืด (Dark Mode)}
ระบบรองรับการแสดงผลในโหมดมืด (Dark Mode) เพื่อถนอมสายตาและประหยัดพลังงาน

\subsubsection{วิธีการใช้งาน}
\begin{enumerate}
    \item สังเกตที่เมนู Sidebar ด้านซ้ายล่าง จะมีปุ่ม \textbf{Theme} พร้อมไอคอนรูปพระจันทร์หรือพระอาทิตย์
    \item กดที่ปุ่ม \textbf{Theme} เพื่อสลับโหมดการแสดงผล
    \item ระบบจะจดจำค่าการตั้งค่าล่าสุดไว้ (เช่น หากเลือก Dark Mode ไว้ เมื่อกลับมาใช้งานใหม่ก็จะยังคงเป็น Dark Mode)
\end{enumerate}

\subsection{การพิมพ์รายงาน (Printable Report)}
ในหน้า \textbf{Reports \& Logs} ผู้ดูแลระบบสามารถพิมพ์รายงานสรุปการใช้งานออกมาในรูปแบบเอกสารกระดาษหรือ PDF ได้อย่างสวยงาม

\subsubsection{วิธีการใช้งาน}
\begin{enumerate}
    \item ไปที่เมนู \textbf{Reports \& Logs}
    \item กดปุ่ม \textbf{PRINT REPORT} ที่มุมขวาบน
    \item ระบบจะแสดงหน้าต่าง Print Preview ของเบราว์เซอร์
    \item สังเกตว่าส่วนที่ไม่จำเป็น เช่น แถบเมนู (Sidebar) และปุ่มกดต่างๆ จะถูกซ่อนไป เหลือเพียงเนื้อหารายงานที่สะอาดตา
    \item เลือกเครื่องพิมพ์หรือเลือก "Save as PDF" แล้วกด Print
\end{enumerate}

\subsection{การค้นหาและแบ่งหน้าข้อมูล (Search \& Pagination)}
ในหน้า \textbf{Customers} (จัดการลูกค้า) ได้เพิ่มระบบค้นหาและแบ่งหน้าเพื่อรองรับปริมาณข้อมูลที่มากขึ้น

\subsubsection{วิธีการใช้งาน}
\begin{enumerate}
    \item \textbf{การค้นหา:} พิมพ์ชื่อ, อีเมล, หรือเบอร์โทรศัพท์ ในช่องค้นหาด้านบนตาราง แล้วกดปุ่ม \textbf{SEARCH} หรือกด Enter ระบบจะแสดงเฉพาะข้อมูลที่ตรงกับคำค้นหา
    \item \textbf{การแบ่งหน้า:} หากมีข้อมูลจำนวนมาก ระบบจะแบ่งแสดงทีละ 10 รายการ สามารถกดปุ่ม \textbf{Prev} หรือ \textbf{Next} ด้านล่างตารางเพื่อเปลี่ยนหน้าได้
\end{enumerate}
